About Us

OKK Code aims to make Coding accessible to all! You have the option to learn the fundamentals of coding at your own pace, with a built in test yourself feature to track your own progress. Start learning to code straight away today, it really is as simple as 1,2,3...!

Despite coming from a psychology background, I have always been interested in technology and have always loved the idea of one day making my own website. However, this meant because of my expertise in psychology, I found it particularly challenging to learn how to code from scratch. All of this inspired us to create OKKCode to enable thse who currently have little knowledge (like I did) to be able to kickstart their own coding adventure!
