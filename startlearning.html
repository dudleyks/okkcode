So Coding...What is it?

Coding or computer programming is used by pretty much anything that uses a plug or battery. It is used to create websites, apps and so much more!


Codes are instructions a computer uses to help it understand the task it is being asked to perform


There are so many languages you can learn! Some examples are HTML, CSS, Javascript, C+, Ruby, Python and the list goes on and on! Each one does different weird and wonderful things!

Let's start with the basics shall we?

Hypertext Markup Language or HTML for short

It is a language that is used to create the layout of a webpage and is the backbone for most of the pages you visit


Most HTML elements have an opening tag < > and a closing tag

The only difference between opening and closing tags is the forward slash after the opening chevron of a closing tag.
